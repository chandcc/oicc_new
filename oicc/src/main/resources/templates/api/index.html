新时代，文化创意新使命

大会以“文化创意赋能”为指导，坚持“筑平台、促改革、创价值”的根本目标，积极推动文化创意＋破解新时代中国的主要矛盾，充分挖掘国内外文化创意设计人才和作品，推动文化创意和设计服务与内蒙古和林格尔新区乃至内蒙古自治区经济社会发展实现对接。

  • 主要征集范围为“一带一路”国家和地区。凡与商贸领域、应用领域、传达领域、数学与科学领域、物质领域等领域产品相关的研发机构、高等院校、品牌企业、设计公司、生产厂商、广告公司以及工艺美术品设计专业人士和创意设计爱好者，均可以单位或个人名义参赛。鼓励国内自主创新、知名品牌企业参赛。

    特别提示：不鼓励获得其它奖项（已超两年）的作品参赛。

    一、价值优先

    健康向上，内涵丰富。具有文化创意＋赋能传统产业的特质。

    二、审美价值

    能够感动心灵。创造性使用优秀文化元素，较好提升产品形象，舒适、美观、协调，具有艺术性、观赏性；色彩搭配合理，形态体现科技与艺术的结合。充分使用本民族的传统文化和技艺丰富创意设计内涵。

    三、创新性

    设计理念具有前瞻性，追求新奇独特，创新点突出。充分利用新科技、新材料，体现创作者的想象力和创造力。

    四、符合绿色环保、可持续发展要求

    在设计、制造、流通、使用、回收全过程注重生态、节能、环保、低碳健康的理念。

    五、工艺精良，品质可靠，材料合理

    具有较高商业价值，创意因素对提高产品附加值和品牌价值贡献较大。

  • 一、作品提交

    每套作品需提供3张彩色效果图和实物照片（精度不低于300dpi.jpg）和200字左右的设计说明。设计图纸和说明文字均不得出现作者的任何信息和记号。（提交参赛作品设计说明，包括设计思想、设计目的、定位、使用方法、知识产权情况等。要求文字精炼，要点清晰。传统工艺需注明其工艺特征。）

    二、报名渠道

    需登录大赛官方网站http://hicc.cnlive.com提交报名作品。

    三、作品产权

    作品不得仿冒已上市的产品或侵犯他人知识产权的产品。所有参赛作品均需做知识产权声明，侵权者将承担给大会造成的影响和损失。参赛者或参赛单位要保证参赛作品不侵犯他人知识产权，如自有知识产权，需要提交相关的权属证书复印件。

    四、投产产品

    投产产品需报送实物。

    五、参赛作品约定

    参赛作品的文字说明材料、照片及实物均不退还本人。报名后不符合参赛要求的作品将通知报送者。

    六、参展声明

    报名后，符合参赛规则的作品所有者应同意参加大赛主办单位的展示会及有关宣传推荐活动。被举办方选定的作品在展览后授权举办方进行IP孵化。

    七、费用说明

    本届评奖免收报名费和评审费。设计费、模型、样品、产品制作费用等均由参赛者自行负担。大件展品的运费由主办方承担。

    • 均为原创作品。结合大赛分类主题特色进行创新，具备新颖的构思和概念，充分考虑新科技、新材料、新工艺的应用，同时强调绿色与环保的理念。设计理念超前，独特新颖，有创新点、有震撼力。充分体现创作者的想象力和创造力，拥有完整的自主知识产权。
